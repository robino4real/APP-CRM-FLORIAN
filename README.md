# CRM React Dashboard

Une application React (Vite) pour transformer vos exports CSV Commandes, Achats, Ventes et Catalogue en un cockpit CRM visuel.

## Démarrage

```bash
npm install
npm run dev
```

## Fonctionnalités clés
- Import CSV pour Commandes, Achats, Ventes, Catalogue (colonnes listées dans l'UI).
- Cartes de synthèse (ventes, achats/commandes, marge, salaire 20% des ventes avec ajustements).
- Graphiques (évolution mensuelle, top fournisseurs, répartition paiements).
- Table ventes avec recherche / filtres.
- Données d'exemple intégrées pour tester sans fichier.
